Auditor Verifier

All cryptography runs entirely in your browser. The server is not trusted for verification — signatures are re-verified client-side using the same SAG Ristretto255 algorithm that produced them. Disconnect your network after the page loads and JSON-paste mode still verifies locally.

All metrics include Laplace noise (ε=1.0). Individual devices cannot be identified. Ring signatures prove someone in the ring attested on this date, not which device.
