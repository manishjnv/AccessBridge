Core Features

Ambient, invisible accessibility that adapts in real-time to user struggles

🧠

Struggle Detection

Real-time behavioral signal analysis: scroll velocity, click accuracy, typing rhythm, cursor path, dwell time, hesitation patterns.

⚙️

Auto Adaptations

Decision engine evaluates struggle scores and automatically applies font scaling, contrast, layout simplification, and more.

🎯

Focus & Reading Mode

Cognitive simplifier with spotlight focus, distraction shield, and reading guide for users with attention difficulties.

🎤

Voice Commands

20+ hands-free commands via Web Speech API: scroll, click, type, navigate tabs, zoom, find text, and more. Hindi supported.

👁️

Eye Tracking

Webcam-based gaze cursor for motor-impaired users. Face-position tracking with 5-point calibration and smooth interpolation.

🖱️

Dwell Click

Auto-click after configurable hover delay. Radial SVG progress indicator, 15px movement threshold, visual pulse feedback.

💤

Fatigue-Adaptive UI

4-level progressive interface simplification based on time-of-day, session duration, and declining interaction rates.

🤖

AI-Powered Insights

3-tier AI engine (local free, Gemini Flash, Claude) for page summarization, text simplification, and readability scoring.

🌍

Multi-Language

Voice commands in Hindi, English, Spanish, French, German, Chinese, Japanese, Arabic. Profile-based language selection.

Architecture

@accessbridge/core

Types, ProfileStore, StruggleDetector, DecisionEngine. Zero dependencies.

@accessbridge/extension

Manifest V3 Chrome extension. React popup, side panel, content scripts, sensory adapter.

@accessbridge/ai-engine

3-tier AI with caching and cost tracking. Summarizer, simplifier, classifier, translator.

Content Modules

Cognitive Simplifier, Voice Commands, Fatigue Adaptive UI, Dwell Click, Eye Tracker, AI Bridge.

Install in Chrome

AccessBridge runs entirely on-device. No server required for core accessibility features.
