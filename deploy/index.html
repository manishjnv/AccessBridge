Real-time behavioral signal monitoring

Ambient Intelligence at Work

AccessBridge continuously monitors subtle behavioral signals to understand user needs. When struggle patterns emerge, the AI decision engine triggers targeted adaptations without requiring any explicit action.

  • Detects motor impairment via click accuracy and cursor tremor
  • Identifies vision struggles from scroll and zoom behavior
  • Adapts layout complexity based on cognitive load signals
  • All processing runs locally -- no data leaves the browser

Core Features

Ambient, invisible accessibility that adapts in real time to user struggles

Struggle Detection

Real-time behavioral signal analysis: scroll velocity, click accuracy, typing rhythm, cursor path, dwell time, and hesitation patterns.

Auto Adaptations

AI decision engine evaluates struggle scores and automatically applies font scaling, contrast boost, layout simplification, and more.

Focus & Reading Mode

Cognitive simplifier with spotlight focus, distraction shield, and reading guide for users with attention difficulties.

Voice Commands

25+ hands-free commands via Web Speech API: scroll, click, type, navigate tabs, zoom, find text, and more. Hindi and 7 other languages supported.

Eye Tracking

Webcam-based gaze cursor for motor-impaired users. Face-position tracking with 5-point calibration and smooth interpolation.

Dwell Click

Auto-click after configurable hover delay. Radial SVG progress indicator, movement threshold detection, visual pulse feedback.

Fatigue-Adaptive UI

4-level progressive interface simplification based on time-of-day, session duration, and declining interaction rates.

AI-Powered Insights

3-tier AI engine (local heuristic, Gemini Flash, Claude) for page summarization, text simplification, and readability scoring.

Multi-Language

Voice commands in Hindi, English, Spanish, French, German, Chinese, Japanese, and Arabic. Profile-based language selection.

Architecture

@accessbridge/core

Types, ProfileStore, StruggleDetector, DecisionEngine. Zero external dependencies.

@accessbridge/extension

Manifest V3 Chrome extension. React popup, side panel, content scripts, sensory adapter.

@accessbridge/ai-engine

3-tier AI with caching and cost tracking. Summarizer, simplifier, classifier, translator.

Content Modules

Cognitive Simplifier, Voice Commands, Fatigue Adaptive UI, Dwell Click, Eye Tracker, AI Bridge.
